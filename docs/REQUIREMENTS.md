## 1. SUPPORTED PLATFORMS

### Included
- Android Phone
- Android Tablet
- Android TV
- iPhone
- iPad

### Excluded (by design)
- Apple TV (tvOS)
- Desktop/Web (future phase)

---

## 2. USER TYPES

### Anonymous Users
Can:
- View Home
- Browse channels
- Search
- Play videos
- View comments
- Store watch history locally

Cannot:
- Upload
- Like / Comment
- Subscribe
- Create playlists
- Save videos
- Report videos
- Access notifications or personal library

Restricted actions must show login prompt.

---

### Logged-in Users
Can:
- Upload videos (mobile/tablet only)
- Replace audio during upload
- Create channel (basic)
- Like / Unlike
- Comment
- Subscribe
- Create playlists
- Save videos
- Report videos
- Access notifications
- Access personal library
- Sync watch history

---

## 3. CORE FEATURE AREAS

- Home & Navigation
